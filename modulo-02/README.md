# Proyecto Apícola Responsivo

Este es mi proyecto para el curso Full Stack Java.

- El proyecto está en la carpeta `modulo-02/proyecto/`.
- Usé HTML, CSS (Flexbox y media queries), imágenes y Bootstrap.
- Se ven productos apícolas y el sitio funciona bien en computador y celular.

## Funcionalidad nueva

Se agregó el archivo `assets/js/app.js` con la función de scroll suave.  
Esto agrega un botón que aparece abajo y al hacer clic te lleva arriba de la página de forma rápida y suave.

## Cómo probar mi trabajo

1. Abrir el archivo `index.html` que está dentro de `modulo-02/proyecto/`.
2. Bajar con el scroll, aparecerá el botón "Ir arriba".
3. Clic en el botón para volver al inicio.

## Evidencias

Las capturas de pantalla para móvil, tablet y escritorio están en la carpeta `assets/img/`.
