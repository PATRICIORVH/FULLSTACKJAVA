# Juego: Adivina el número (1-10)

Es un juego donde el computador elige un número al azar entre 1 y 10 y la persona o usuario tiene 3 intentos para adivinarlo.

## Archivos

- `index.html`: estructura básica de la página.
- `app.js`: lógica del juego en JavaScript.

## Cómo usarlo

1. Se abre `index.html` en el navegador.
2. Cuando aparezca el mensaje, se escribe un número entre 1 y 10.
3. Hay 3 intentos.  
   - Si se acierta, el juego avisa que ganaste.  
   - Si no se acierta, se muestra cuál era el número correcto. 
   - En la página se puede ver el listado de los números que se han intentado.
