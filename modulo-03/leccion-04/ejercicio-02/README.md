# Ecuación de segundo grado

Este ejercicio resuelve una ecuación de segundo grado de la forma `ax² + bx + c = 0` usando JavaScript.

## Archivos

- `index.html`: página básica donde se muestran los resultados.
- `app.js`: código en JavaScript con la función del discriminante y las soluciones.

## Cómo usarlo

1. Abrir `index.html` en el navegador.
2. Ingresar los valores de `a`, `b` y `c` cuando lo pida la pantalla.
3. El programa calcula el discriminante y:
   - Si hay dos soluciones reales, las muestra.
   - Si hay una sola solución, la muestra.
   - Si no hay soluciones reales, avisa al usuario.

## Los resultados se ven:
- En una ventana de alerta.
- En la consola del navegador.
- En el texto de la página, dentro del bloque de resultado.
