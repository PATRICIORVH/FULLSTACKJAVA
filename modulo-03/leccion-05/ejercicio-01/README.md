# Conteo de Vocales
 
Este ejercicio cuenta el total de vocales en un grupo de palabras que se ingresa.

## Archivos

- `index.html`: página web donde se ven los resultados.
- `app.js`: código JavaScript con la función para contar vocales.

## Cómo usar

1. Se abre `index.html` en el navegador.
2. Se ingresa la cantidad de palabras que se desea analizar.
3. Se escribe cada palabra cuando el programa lo pida.
4. El programa calcula cuántas vocales hay en total y muestra el resultado en:
   - Una ventana de alerta.
   - La consola del navegador.
   - Un texto en la página.

## Ejemplo

- Cantidad de palabras: 3
- Palabras: "Hola", "Mundo", "AEIOU"
- Resultado: Total de 9 vocales
