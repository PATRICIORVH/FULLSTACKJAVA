# Ejercicio 02 - Gestión de Pacientes

Este ejercicio practica JavaScript básico usando listas y objetos para guardar datos de pacientes.

## ¿De qué se trata?

La idea es simular un consultorio.  
La página pide cuántos pacientes se van a atender y luego solicita estos datos para cada uno:

- Nombre
- Edad
- Peso en kilos
- Sexo (M/F)

Con esta información, el programa:

- Guarda cada paciente como un objeto dentro de una lista (arreglo).
- Recorre la lista y calcula el **promedio de peso** de todos los pacientes.
- Muestra el resultado en la página y también en la consola del navegador.

## ¿Qué se aprende con este ejercicio?

- Crear y usar arreglos en JavaScript.
- Crear objetos para ordenar la información (diccionarios clave–valor).
- Escribir funciones que reciben datos y retornan un resultado.
- Usar la consola del navegador para revisar mensajes y resultados.

## Cómo ejecutar el proyecto

1. Abrir la carpeta `ejercicio-02` en Visual Studio Code.  
2. Abrir el archivo `index.html` en el navegador (doble clic o usando una extensión como Live Server).  
3. Ingresar la cantidad de pacientes cuando aparezca el primer mensaje.  
4. Completar los datos que se piden para cada paciente.  
5. Ver el **promedio de peso** en la página y revisar el detalle de cada paciente en la consola del navegador.

## Tecnologías usadas

- **HTML** para la estructura básica de la página.  
- **JavaScript** para pedir los datos, guardarlos y calcular el promedio de peso.
