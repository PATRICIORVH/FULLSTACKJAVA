# Ejercicio en JavaScript para calcular el área de un círculo:

Se pide el diámetro con un cuadro de diálogo.

Calcular el radio y el área usando la fórmula A = π * r².

Mostrar el resultado en una alerta, en mensajes del navegador y también en la página.

## Los archivos son:

index.html, que es laestructura básica de la página.

app.js, que es el código JavaScript con el cálculo y las salidas.
