# Ejercicio 01 - Promedio ponderado del curso

En este ejercicio el programa calcula el promedio final de un curso usando tres notas por cada alumno. Primero pide al usuario cuántos alumnos tiene el curso con `prompt()`.

Luego, con un ciclo `for`, pide las 3 notas de cada alumno y aplica estas ponderaciones:
- Nota 1: 25%
- Nota 2: 35%
- Nota 3: 40%

Con esas ponderaciones calcula el promedio de cada alumno y lo va sumando en una variable llamada `notasCurso`. Al final divide esa suma por la cantidad de alumnos para obtener el promedio final del curso.

El resultado se muestra en la página usando `document.write()`.
