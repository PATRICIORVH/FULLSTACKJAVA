# Ejercicio 02 - Mayor y menor con burbuja

Este ejercicio pide 3 números usando `prompt()`, los guarda en un array y los ordena con el método de burbuja usando un ciclo `do-while`.

Después de ordenar, el programa toma el menor número en la posición `0` del array y el mayor en la última posición. Si los tres números son iguales, muestra un mensaje indicando que todos son idénticos. El resultado se muestra en la página con `document.write()`.
